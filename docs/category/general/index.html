General

Reference list of general modules used in-game.

📄️ Replica

Author: loleris\

📄️ Animator

Animator - is an abstraction class which allows for much easier animation in Roblox.

📄️ AttackingAiSpawner

AttackingAiSpawner (AAS) - no description.

📄️ BTreeCreator

Author: tyridge77\

📄️ BehaviorBuilder

BehaviorBuilder - module for creating, setuping and attaching behavior tree's to Characters.

📄️ FXReplicatorService

FXReplicatorService - module for replicating effects across clients.

📄️ FastCastRedux

Author: etithespir

📄️ Fusion

Author: elltob

📄️ Gear

Gear - base Class for Tool objects that player can equip and interact with. Works both Server/Client.

📄️ PathPlus

PathPlus - abstraction module for PathfindingService allowing for fast implementation to the pathfinding NPC's.\

📄️ PhysicsProjectile

📄️ PlayerManager

PlayerManager - module which manages currently connected users on the server. Includes a collection of utility functions.

📄️ Ragdollify

Ragdollify - module used for creating ragdolls for Characters.

📄️ SoundReplicator

SoundReplicator - no description.

📄️ StunPlayer

StunPlayer - module that lets you stun player character.

📄️ Throwable

Throwable - class that allows you to create throwable tool objects with ease.

📄️ ToolController

ToolController - an abstraction layer for tool instance which adds more input events.

📄️ Warp

Author: imezx

Author: evaera
