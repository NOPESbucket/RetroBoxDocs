Ragdollify

Ragdollify - module used for creating ragdolls for Characters.

Properties​

> Character: Model Public​

Reference to character who will be ragdolled.

> Humanoid: Humanoid Public​

Reference to humanoid of the character.

> Player: Player? Public​

Reference to a player. May be nil of ragdoll was created on the NPC.

> Unragdolled: Signal.Signal<nil> Public​

Fires when :Unragdoll() is called.

> Ragdolled: Signal.Signal<nil> Public​

Fires when :Ragdoll() is called.

> Touched: Signal.Signal<BasePart> Public​

Fires when ragdoll's body part touches new part.

> TouchEnded: Signal.Signal<BasePart> Public​

Fires when ragdoll's body part stop's touching touched part.

> _trove: Trove.Trove Private​

> _unragdollTimer: Timer.Timer Private​

> _torso: BasePart Private​

> _isRagdoll: BoolValue Private​

Constructor​

> Ragdollify.new(character: Model)​

Methods​

> Ragdollify:Ragdoll() Public​

Forces character enter ragdoll.

> Ragdollify:Unragdoll(unragdollOnTouch: boolean?, timeout: number?) Public​

Forces character exit ragdoll.

> Ragdollify:IsRagdolled(): boolean Public​

Returns boolean of whether character in ragdoll state.

> Ragdollify:_attachListener() Private​

Attaches collision listeners to the characters body part for tracking collisions of the character.

> Ragdollify:_replaceJoints() Private​

Replaces every single bone (Motor6D) inside Character.

> Ragdollify:_resetJoints() Private​

Destroys all Ragdoll made instances and re-enables the Motor6D's.

> Ragdollify:Destroy() Public​

Destroy's the Ragdoll, releasing it from memory.
