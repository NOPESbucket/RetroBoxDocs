Animator

Animator - is an abstraction class which allows for much easier animation in Roblox. Instead of trying to find the animator inside the character and load the animations individually, Animator allows you to pass in the character, load animations in batch and play them by name. This streamlines the process of adding animation in-game.

Properties​

> Character: Model Public​

Reference to Character model that is animated.

> Animator: Animator Public​

Reference to Animator of the Character.

> CurrentAnimation: AnimationTrack Public​

Reference to a currently played animation track.

> AnimationRegistry: { [string]: AnimationTrack } Public​

A registry which contains every loaded animation.

> AnimationIdRegistry: { [string]: string } Public​

A registry which contains every ID of loaded animation.

> Started: Signal.Signal<string> Public​

Fires whenever animation started playing through :Play() method of the AnimatorObj.

> Stopped: Signal.Signal<string> Public​

Fires whenever animation ended (but not interpolated to default position) it's playback or when :Stop() method is called.

> Ended: Signal.Signal<string> Public​

Fires whenever animation completelly ended it's playback.

> MarkerReached: Signal.Signal<Player, string> Public​

Fires whenever marker is reached in the animation playback, allowing to bind specific behavior to the animation.

> SoundPlayed: Signal.Signal<Sound> Public​

Fires whenever sound was played in the animation playback, allowing to bind specific behavior to the animation.

> Destroying: Signal.Signal<nil> Public​

Fires whenever AnimatorObj is being destroyed via :Destroy() method.

> _trove: Trove.Trove Private​

Constructor​

> Animator.new(character: Model, animations: { Animation })​

Method​

> Animator:GetCurrentAnimation(): AnimationTrack? Public​

Returns currently played animation. May not return AnimationTrack if animation isn't playing.

> Animator:_loadAnimation(animation: Animation): boolean Private​

Loads the animation inside the AnimatorObj allowing it to be played via :Play().

> Animator:StopAll() Public​

Stop's every animation that's currently playing in the AnimatorObj.

> Animator:Play(animationName: string) Public​

Play's the animation.

> Animator:Stop(animationName: string) Public​

Stop's the animation.

> Animator:ConnectMarkerReachedSignal(animationName: string, markerName: string, callback: () -> ()): RBXScriptConnection? Public​

Connects a function callback to the animation with the specified marker. May not return RBXScriptConnection if invalid names was passed.

> Animator:GetAnimation(animationName: string): AnimationTrack? Public​

Returns the AnimationTrack. May not return AnimationTrack if invalid names was passed.

> Animator:GetAnimationAssetId(animationName: string): string? Public​

Returns the animation AssetId. May not return AssetId if invalid names was passed.

> Animator:IsPlaying(animationName: string): boolean Public​

Return boolean of whether animation is playing currently or not.

> Animator:AdjustSpeedAndPlay(animationName: string, speed: number) Public​

Play's the animation with the specified speed.

> Animator:Destroy() Public​

Destroy's the AnimatorObj, releasing it from memory.
