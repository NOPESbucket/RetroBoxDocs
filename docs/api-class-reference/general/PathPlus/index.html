PathPlus

PathPlus - abstraction module for PathfindingService allowing for fast implementation to the pathfinding NPC's.
Includes usefull features such as:

  • Pathfinding Debug mode
  • Unstuck mechanism
  • Non-reliant to Humanoid events

Types​

type AgentParameters = {
	AgentRadius: number?,
	AgentHeight: number?,
	AgentCanJump: boolean?,
	AgentCanClimb: boolean?,
	WaypointSpacing: number?,
	Costs: { [string]: number }?,
}

For more info what each parameter does, go to Roblox documentation here.
